# Docs

Tài liệu trong thư mục này được gom theo mục đích để root `docs` không bị lẫn plan, reference và ghi chú điều tra.

## Cấu trúc

- `plans/`: kế hoạch triển khai theo domain.
- `reference/`: tài liệu tham khảo, hướng dẫn vận hành, context dự án.
- `investigations/`: ghi chú điều tra sự cố hoặc phân tích nguyên nhân.
- `wallets/`: tài liệu chuyên sâu/legacy cho wallet.
- `wallet-issues/`: task, solution và handoff theo issue wallet.
- `wash-trading/`: hướng dẫn, quick start, reference và troubleshooting cho wash trading.
